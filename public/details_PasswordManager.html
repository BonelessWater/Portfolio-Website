Project information

  • Project URL: github.com/BonelessWater/Password-Manager

Inspiration

With the increasing number of online accounts and the constant threat of security breaches, I wanted to create a password manager that combines the convenience of modern biometric authentication with robust security practices. The goal was to eliminate the need to remember dozens of complex passwords while maintaining the highest security standards.

Explanation

This cross-platform desktop password manager leverages Spring Boot for the backend REST API, CEF for the desktop UI, and MySQL for encrypted credential storage. The standout feature is Face ID authentication using OpenCV, which achieves 98% biometric verification accuracy. The Java backend orchestrates data flow between biometric services, password vaults, and the user interface, reducing authentication time by 40%. The application provides seamless credential management across devices while ensuring all sensitive data is encrypted both in transit and at rest.

Takeaways

This project taught me the complexities of building secure, cross-platform applications. I gained deep experience with biometric authentication systems, learning how to handle edge cases like varying lighting conditions and facial changes. Integrating multiple technologies—from OpenCV's computer vision to Spring Boot's REST framework—challenged me to think about system architecture holistically. The experience reinforced the importance of balancing security with user experience, ensuring that robust protection doesn't come at the cost of usability.
